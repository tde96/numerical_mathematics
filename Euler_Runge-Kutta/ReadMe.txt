Für die Programmieraufgabe 02 im Fach Numerische Mathematik wurden ein Programmcode für das Euler und Runge-Kutta Verfahren entwickelt.Da es zu Problemen beim Hochladen der .exec Datei kommen kann,
Wurden zusaetzlich alle Dateien in eine .zip Datei komprimiert.Folgende Daten wurden erstellt:

1. create_plot.m 		Code für die Visualisierung der berechneten Werte
2. main.cpp 			Quelltext der Berechnung in C++
3. main.txt			Berechnung-Code als .txt Datei4. nm_2_werte.csv		Berechnete Werte
5. num_mathe_projekt_2		Ausfuerbare Unix-Datei
6. plot.jpg			Plot der Funktionen des Euler und Runge-Kutta Verfahrens
